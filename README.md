# Android-Message-Reconstruction
A Python script that extracts SMS and MMS conversations from an Android Messages mmssms database and exports each conversation thread into a separate text file under a 'Message Threads' folder.
## Features
- Reads SMS messages from Android message databases
- Extracts MMS content and associated sender information
- Converts timestamps into readable dates
- Organizes messages by conversation thread
- Exports each thread as a separate text file under a 'Message Threads' folder
## Requirements
- Python 3.x
- SQLite3
## Usage
python AndroidMessages.py
